# Namazım v1.0.0

İlk public sürüm yayınlandı.

## Öne Çıkanlar

- Namaz vakitleri + canlı geri sayım
- Ramazan/takvim ekranı
- Günün hadisi + koleksiyonlar
- Bildirim ve alarm ayarları
- Konumdan otomatik şehir/ilçe
- Kıble pusulası ve kıble haritası

## Teknik Notlar

- Build: `./gradlew :app:assembleDebug`
- APK: `app-debug.apk` (release asset olarak eklenecek)

## Bilinen Not

- Harita ekranı için geçerli Google Maps API key gereklidir.
